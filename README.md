# ada-desafio-tecnico-fullstack

## Get started

### Backend
1 - copy .env.example  
`cp .env.example .env`  
2 - start docker  
`docker-compose up -d `  
3 - get current version of node
`nvm install 18`  
`nvm use`  
4 - install dependences  
`yarn install `  
5 - run application   
`yarn run dev `
